icat.table.TableContentsTemplate#

class icat.table.TableContentsTemplate(**kwargs)#

Bases: VuetifyTemplate

The primary table used in the DataManager class. This is a heavily modified vuetify DataTable, largely necessary for us to be able to put a more comprehensive set of action buttons in each row.
